## Prerequisites

Before installing XDK, ensure you have:

- **Python 3.8 or higher** installed on your system
- **pip** or **uv** package manager
- An X developer account with API credentials (see [X Developer Portal](https://developer.x.com))

## Install XDK

Choose your preferred package manager:

<CodeGroup>
```bash pip
pip install xdk
```

```bash uv
uv add xdk
```
</CodeGroup>

<Note>
The current version is **0.9.0**. This SDK is auto-generated from the X API OpenAPI specification.
</Note>

## Verify Installation

Confirm XDK is installed correctly:

```python
import xdk

print(f"XDK version: {xdk.__version__}")
# Output: XDK version: 0.9.0
```

## Get API Credentials

To use XDK, you'll need credentials from the [X Developer Portal](https://developer.x.com). The type of credentials depends on your authentication method:

<Tabs>
  <Tab title="OAuth 2.0 PKCE (Recommended)">
    Best for apps that act on behalf of users with automatic token refresh.
    
    **Required credentials:**
    - Client ID
    - Client Secret (for confidential clients)
    - Redirect URI (configured in your app settings)
    
    **How to get them:**
    1. Go to the [X Developer Portal](https://developer.x.com)
    2. Create or select your app
    3. Navigate to "Keys and tokens"
    4. Find your **Client ID** under "OAuth 2.0 Client ID"
    5. For confidential clients, generate a **Client Secret**
    6. Configure your **Redirect URI** in app settings
  </Tab>
  
  <Tab title="OAuth 1.0a">
    Traditional OAuth flow for user-context authentication.
    
    **Required credentials:**
    - API Key (Consumer Key)
    - API Secret (Consumer Secret)
    - Access Token
    - Access Token Secret
    
    **How to get them:**
    1. Go to the [X Developer Portal](https://developer.x.com)
    2. Create or select your app
    3. Navigate to "Keys and tokens"
    4. Copy your **API Key** and **API Secret**
    5. Generate **Access Token** and **Access Token Secret**
  </Tab>
  
  <Tab title="Bearer Token">
    Simple app-only authentication for read-only operations.
    
    **Required credentials:**
    - Bearer Token
    
    **How to get it:**
    1. Go to the [X Developer Portal](https://developer.x.com)
    2. Create or select your app
    3. Navigate to "Keys and tokens"
    4. Copy or regenerate your **Bearer Token**
    
    <Warning>
    Bearer tokens provide read-only access and cannot be used for write operations like creating posts.
    </Warning>
  </Tab>
</Tabs>

## Environment Setup

Store your credentials securely using environment variables:

<CodeGroup>
```bash .env
# OAuth 2.0 PKCE
X_CLIENT_ID="your_client_id"
X_CLIENT_SECRET="your_client_secret"
X_REDIRECT_URI="http://localhost:3000/callback"

# OAuth 1.0a
X_API_KEY="your_api_key"
X_API_SECRET="your_api_secret"
X_ACCESS_TOKEN="your_access_token"
X_ACCESS_TOKEN_SECRET="your_access_token_secret"

# Bearer Token
X_BEARER_TOKEN="your_bearer_token"
```

```python Load with python-dotenv
import os
from dotenv import load_dotenv

# Load environment variables
load_dotenv()

# Access credentials
client_id = os.getenv("X_CLIENT_ID")
client_secret = os.getenv("X_CLIENT_SECRET")
```
</CodeGroup>

<Tip>
Install `python-dotenv` to easily load environment variables:
```bash
pip install python-dotenv
```
</Tip>

## Initialize the Client

Once installed, you can initialize the XDK client with your preferred authentication method:

<CodeGroup>
```python OAuth 2.0 PKCE
from xdk import Client
import os

client = Client(
    client_id=os.getenv("X_CLIENT_ID"),
    client_secret=os.getenv("X_CLIENT_SECRET"),
    redirect_uri=os.getenv("X_REDIRECT_URI")
)

# Get authorization URL for user login
auth_url = client.get_authorization_url()
print(f"Visit: {auth_url}")

# After user authorizes, exchange code for tokens
# code = "..." # from callback
# token = client.exchange_code(code)
```

```python OAuth 1.0a
from xdk import Client
from xdk.oauth1_auth import OAuth1
import os

auth = OAuth1(
    api_key=os.getenv("X_API_KEY"),
    api_secret=os.getenv("X_API_SECRET"),
    access_token=os.getenv("X_ACCESS_TOKEN"),
    access_token_secret=os.getenv("X_ACCESS_TOKEN_SECRET"),
    callback="oob"  # or your callback URL
)

client = Client(auth=auth)
```

```python Bearer Token
from xdk import Client
import os

client = Client(
    bearer_token=os.getenv("X_BEARER_TOKEN")
)

# Note: Bearer token only supports read operations
```

```python Direct Access Token
from xdk import Client
import os

# If you already have an OAuth2 access token
client = Client(
    access_token=os.getenv("X_ACCESS_TOKEN")
)
```
</CodeGroup>
